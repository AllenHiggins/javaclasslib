# MaxMin
List of numbers (int) that return the max and min values.

Input {1,2,3,4,5} Output: String "14 10"

# MaxMin Example:

2+3+4+5 = 14

1+3+4+5 = 13

1+2+4+5 = 12

1+2+3+5 = 11

1+2+3+4 = 10

Max = 14, Min = 10
